# 42-ft_container
### What is STL Library?
	STLとは、動的配列や双方向リンクリスト、連想配列などといった基本的なデータ構造を、type-safeかつ最小コーディングで再利用するためのライブラリ。
	ライブラリとはいっても、ほとんどtemplateでできているので、ヘッダファイルをインクルードすればすぐ使えます。
